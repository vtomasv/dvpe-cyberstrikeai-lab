# API / Referencia (Ejemplo)

## HTTP `/api/skills/{id}`

- `resource_path`: ruta relativa dentro del paquete, por ejemplo `FORMS.md`, `scripts/payloads.txt`.
- `depth=summary`: solo resumen y directorio, adecuado para la primera ronda de búsqueda.

## Tiempo de ejecución multiagente

- Usa la herramienta **`skill`** de Eino ADK para cargar progresivamente por paquete de habilidades; opcionalmente activa `multi_agent.eino_skills.filesystem_tools` para acceder a los archivos dentro del paquete.
